# Student Management System

A console-based Student Management System developed using Python.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student

## Student Details Stored

* Student ID
* Student Name
* Section
* Email
* Phone Number
* College Name

## Technologies Used

* Python
* Lists
* Dictionaries
* Loops
* Conditional Statements

## How to Run

```bash
py StudentManagementSystem.py
```
